

# Cocktail API


This is API to get cocktail receipt using ASP.NET 7 and the data is wrapping from CocktailDB

## Documentation
| URL | Method | Param|
|--|--|--|
|api/Cocktail/page/{page}  | GET |page: int|

|api/Cocktail/ | GET |-|
|api/RandomCocktail  | GET |-|
